    hornbeam_pubsub (hornbeam v1.0.0)

    View Source

    Pub/Sub messaging for Python apps.

    This module provides publish/subscribe messaging backed by Erlang's pg (process groups). It enables: - Real-time updates to web clients - Event broadcasting - Cross-node messaging (distributed pg)

    Note: This is primarily for Erlang-side subscriptions. For Python web handlers, use WebSocket connections with topic routing.
