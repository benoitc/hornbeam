    Distributed Erlang RPC for Python apps.

    This module lets Python apps call functions on remote Erlang nodes. Use cases: - Distributed ML inference across GPU nodes - Sharding data processing across cluster - Calling specialized services on different nodes

    Erlang distribution provides: - Transparent RPC (call any node in cluster) - Node discovery and monitoring - Fault-tolerant (handle node failures)
